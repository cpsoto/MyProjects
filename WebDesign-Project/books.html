My Books

This web site contains several descriptions of some of my favorite books. I created this web site to give you some basic overviews of the books that I consider to be not only personal favorites of mine, but also entertaining. I hope you will like my book descriptions and furthermore, I hope you will purchase one of my books from Amazon. Below is my favorite book.

This C Programming Language By Dennis M. Ritchie and Brian W. Kernighan is an excellent books for both new comers and expert programmers. The book is packed with many examples. and is yet small and concise. This book is written by the person that created the language, so there is no better source of information. I highly recommend this book to anyone who wishes to become a C programmer.

HTML and CSS is a great book for new comers to web Design. This book is packed with lots of examples, and is easy to read for beginners and challenging enough for experienced programmers and web designers. In this updated edition author Bruce Hyslop uses crystal-clear instructions and friendly prose to introduce you to all of today's HTML and CSS essentials. It is a must read.

Database Systems: The Complete Book is ideal for Database Systems and Database Design and Application courses offered at Computer Science departments. Definitely recommended for beginner student's of Computer Sciences interested in Database concepts and applications A basic understanding of laws, basic data structures and programming environments. Click on image to buy.

Head First Java is a very good book for newcomers to the world of OOP and Java. The book is very engaging containing many images, puzzles and novel ways of introducing materials which may be difficult to grasp otherwise. Content includes basic OOP concepts, java rules, advanced topics, threads, networks and queues among other topics. To learn more about this book click image.
